Продающая статья-история от лица клиента — мощный инструмент доверия.

**Формат:** История реального человека, который решил проблему с помощью вашего продукта.

## Что делает этот промпт

| | | |

| Показывает результат | Перечисляет преимущества | Создаёт доверие |

## Для лучшего результата

- Вы опишете характеристику автора (возраст, профессия)
- Расскажете о проблеме, которую он решил
- Дадите характеристики продукта
- Укажете преимущества
