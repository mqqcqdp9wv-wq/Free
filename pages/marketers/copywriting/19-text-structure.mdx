# 19. Структура текста

Создание плана статьи перед написанием — первый шаг к качественному тексту.

<Callout type="info">
**Начните с плана** — это сэкономит время и сделает текст логичным.
</Callout>

## Что делает этот промпт

| | | |
|:--|:--|:--|
| Создаёт структуру статьи | Формулирует основную идею | Разбивает на подразделы |
| Пишет тезисы для каждой части | Готовит вводную часть | Продумывает заключение |

## Для лучшего результата

- Вы укажете тему статьи
- Опишете целевую аудиторию
- Скажете, какой результат хотите от текста
