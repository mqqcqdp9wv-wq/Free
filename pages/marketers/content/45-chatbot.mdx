# 45. Структура чат-бота

Разработка логики и сообщений для чат-бота с квалификацией лидов.

<Callout type="info">
**Квалификация через бота:** Вопросы → Телефон → Материал. Автоматический сбор лидов.
</Callout>

## Что делает этот промпт

| | | |
|:--|:--|:--|
| **Создаёт приветствие** — первое впечатление | **Пишет 4-5 вопросов** — квалификация лида | **Мотивирует отвечать** — вовлечение |
| **Запрашивает телефон** — ключевой контакт | **Выдаёт ценность** — награда за ответы | **Продумывает логику** — последовательность |

## Для лучшего результата

- Вы опишете услугу
- Укажете, какой материал выдаёте
- Расскажете, какие вопросы важны для квалификации
