**Результат:** Готовый к передаче дизайнеру документ с описанием каждого экрана.

## Что делает этот промпт

| | | |
|:--|:--|:--|
| Анализирует боли и возражения ЦА | Определяет конверсионную стратегию | Расписывает каждый экран детально |

## Для лучшего результата

- Вы подробно опишете бизнес и продукт
- Укажете тип бизнеса (B2B/B2C)
- Перечислите страхи и возражения ЦА
