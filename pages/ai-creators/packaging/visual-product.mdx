# Визуальная упаковка товара

## Что делает этот промт

| | | |
|:--|:--|:--|
| Помогает визуально показать продукт так, будто он уже существует | Позволяет менять фон, ракурс, интерьер, материал, свет | Даёт фоновый рендер без расходов на фотосессию |
| Упрощает генерацию товаров под Wildberries, Etsy, Amazon, Ozon | Позволяет тестировать визуальный стиль бренда | Работает с любыми объектами: мебель, одежда, косметика, еда |
| Экономит деньги на 3D-визуализации, съёмках и мокапах | | |

## Будет классненько, если

- Ты дашь фото товара или опишешь его максимально чётко
- Уточнишь, для какой платформы делается визуал (WB, Etsy, лендинг и т.д.)
- Скинешь пару визуальных референсов: «вот такой фон, вот такой свет, вот так подан товар»
- Напишешь, какой стиль точно не подходит: «не глянцево», «не по-русски», «не молодёжно», «без пустоты»
- Опишешь, какая эмоция должна быть: «хочется потрогать», «уютно», «природно, но минималистично», «дизайнерская эстетика»
