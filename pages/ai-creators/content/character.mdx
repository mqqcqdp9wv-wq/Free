# Симуляция героя / персонажа

Создание сильного, узнаваемого голоса для коммуникации.

## Что делает этот промт

| | | |
|:--|:--|:--|
| Разрабатывает голос и образ персонажа | Помогает вырасти над собой и не постесняться узнаваемым голосом | Раздвигает границы: через маску говорить честнее |
| Делает контент живым, через «чужого персонажа» | Комментирует персонажи различных архетипов: наставник, гопник, маг | Строит голос, который работает не только раз, а постоянно |
| Определяет тон, стиль, фразы, интонации | Помогает создавать автоответы, прогревы, курс через голос героя | |

## Будет классненько, если

- Вы реально боитесь говорить от себя — но если говорить от маски, это проще
- У вас есть вайб, но вы не можете его оформить в голос
- Вы хотите смелости говорить так, как не позволяли себе ранее
- Вы готовы примерить роль: бунтарь, наставник, тихий, странный
- Вы понимаете, что через персонажа удобнее доносить правду, чем от лица «продающего эксперта»
