# No-Code / AI-разработка

Промпты для создания приложений без написания кода.

## Что такое Vibecoding?

**Vibecoding** — это новый подход к разработке, где вы описываете идею словами, а AI генерирует готовый код.

**Как это работает:**

1. Вы пишете краткое описание приложения (бриф)
2. AI генерирует техническое задание (PRD)
3. Vibecoding-инструмент (bolt.new, Cursor, Replit) создаёт рабочий код
4. Вы получаете готовое приложение

## Инструменты

- **bolt.new** — генерация веб-приложений из описания
- **Cursor** — AI-редактор кода
- **Replit** — онлайн-среда с AI-помощником
- **v0.dev** — генерация UI-компонентов
